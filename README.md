主要内容：输入输出字符串

- `print()`输出控制
- 变量：
  - 变量赋值；
  - 变量值互换（一句）
- 字符串：
  - 单、双、三引号用法
  - 转义符“\”用法（转义、行末未完待续）
  - 原始字符串“r”
  - 长字符串
  - 字符串加乘法

### 02 基础函数

- 函数`input()`
- 模块`random`
  - `random.randint(x,y)`
  - 随机数重现
    - `random.getstate()`
    - `random.setstate()`
- 小程序示例：应用 分支语法，比较运算符，算数运算符，循环语法，嵌套，random 模块
- 数值类型
  - 整型
  - 浮点型
    - 模块`decimal`：十进制定点和浮点运算，解决浮点型精度问题
  - 复数
- 数值计算
  - 数值计算的类型
    - 运算符`+`,`-`,`*`,`/`,`//`,`%`,`-x`,`+x`,`**`
    - 运算函数`divmod()`,`abs()`,`pow()`,`c.conjugate()`
    - 类型转换函数`int()`,`str()`,`float()`,`complex()`
- 布尔类型
  - `Bool()`：返回布尔值`T`or`F`
    - `Bool()`返回`False`的情况
  - 逻辑运算符（布尔运算）
    - `and`,`or`,`not`
    - `True` = 1, `False` = 0
    - `and`和`or`运算
      - 操作数
      - **短路逻辑**
- 运算符优先级
